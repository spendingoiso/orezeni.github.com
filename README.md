# 俺のゼニはどこ行った for 大磯町

wheredoesmymoney.orgの大磯町バージョンです．

http://spending.jp/ で稼働しているサイトのソースコードです。

Where Does My Money Go? β版（2013年10月22日稼働）は、大磯町民が大磯町に納めている市税を対象として構築したものです。自分の年間総収入をスライドで設定し、単身世帯か扶養一人世帯かを選択すると、給与所得者であるという前提で、大磯町に納めている町税年総額と14分野毎に一日当たり支払っている市税額が表示されます。所得控除額は、単身世帯の場合33万円、扶養一人世帯の場合66万円、市税負担額は課税対象額の6％と設定しています。また、14分野への税金の配分比率は、大磯町の一般会計全体における14分野への予算配分比率(平成25年4月1日現在)を使っています。

このサイトは、軽い気持ちで

立堀 道昭(Michiaki Tatsubori), @tatsubori

によって作られましたが、元の spending.jp をベースにちょっとカスタマイズしただけです。

元のサイトは、公共データのオープン化によって、地域社会を変え、日本社会を変え、世界にも影響を与え、貢献して行こうという思いを持って集まった以下のメンバーによって構築、運営されています。
